# ✈️ Airline Customer Satisfaction Dashboard

## 📌 Overview

This project analyzes airline passenger data to evaluate customer satisfaction, service performance, and operational efficiency. The dashboard provides actionable insights to improve customer experience.

## 🎯 Objectives

* Analyze satisfaction by customer type, class, and travel purpose
* Identify key factors affecting passenger satisfaction
* Evaluate impact of delays on customer experience
* Provide business recommendations

## 📊 Dashboard Features

* Overall satisfaction analysis
* Customer segmentation (Loyal vs Disloyal)
* Service quality ratings (WiFi, seat comfort, cleanliness)
* Delay analysis (departure & arrival)
* Demographic insights (age, gender)

## 🛠️ Tools & Technologies

* Power BI
* SQL
* Python (Pandas)
* Excel

## 📁 Dataset

* Airline passenger dataset (25,000+ records)

## 📌 Key Insights

* Business class passengers show higher satisfaction
* Delays significantly impact customer ratings
* Inflight services strongly influence overall experience

## 🚀 How to Use

1. Open the Power BI dashboard (.pbix file)
2. Use filters (class, travel type, customer type)
3. Explore interactive visualizations
